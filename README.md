# Frequently traded items tab

A Plugin that adds a bank tab with items that are found to be frequently traded, based on locally logged trades via the GE.
Items can be manually included/excluded via the configurations, along with setting a threshold for frequently traded items.
Furthermore, the prices are displayed for placeholder items included in the bank tab, even if there is no quantity of items in the bank.
Frequently traded items are defined via plugin configurations and cached trades. Although trades are cached separately per account, 
frequently traded items are based on merged trade history.
